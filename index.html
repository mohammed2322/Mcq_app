Deployment steps

  1. Configure .env (DATABASE_URL, NEXTAUTH_SECRET, GOOGLE_ID).
  2. Run prisma migrate and seed data (50 questions / 5 categories).
  3. Build Docker image and deploy via container registry.
  4. Enable CDN caching and image optimization.
